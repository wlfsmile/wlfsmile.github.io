浮动

  • 浮动目的：最初是为了其他内容（如文本）“围绕”该图像，后来CSS允许浮动任何元素
  • 浮动产生bug的原因：当一个内层元素浮动时，如果没有关闭浮动，父元素就不会再包含这个浮动的内层元素了，因为此时浮动元素已经脱离了文档流，导致外层不能被撑开
